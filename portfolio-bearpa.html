产品主体形象传达活力趣味，有力量的精神，中性化

LOGOTYPE

同时结合熊熊IP打造搞怪娱乐化，性别普适性强的APP形象
FONTS
思源黑体
COLORS
ICONS
#主界面
首页 - 直播入口
主页直播入口的设计，突出互动游戏交互模式选择
精简卡片信息，为用户简化选择，便于快速进入游戏互动
卡片式设计
突出游戏直播的内容、模式和主播，引导用户
滑动切换多种直播模式
信息简化
主页内容简化，除去兀余信息，突出用户个人
特色进行推荐，精选直播频道
[ 社区 ]
加入话题形式，在直播后仍能在主播与观众之间形成“强力社交圈”
通过话题引入相应活动，促进观众加入游戏互动
[ 关注 ]
关注-直播模块快速进入关注主播频道快捷常用，不费时
关注-友圈模块及时抓住主播动态，不错过每一场直播互动
熊熊——你的最佳直播伴侣
IP形象设计-选择熊这个动物元素，体现憨厚但又有力量的形象，符合游戏互动的特性。整体造型雄壮可爱又有些调皮，体现互动的娱乐性，同时侧重年轻群体，性别普适性强
虚拟宠物
陪伴用户并加以装扮定制类功能，同时作为
角色参与用户的体验互动中，收获奖励
等级载体
作为用户等级成长的载体，提升用户粘性
体现主播能力同时反应观众参与度
记录时光
将用户互动情况记录在熊熊模块，互动战果
录互动奖励，足迹查看互动记录
[ 我的 ]

加入快速通道的功能，能够直接进入用户最常观看的直播
提供每周互动记录，并加以周报的形式提升用户掌握度
#直播模块
「互动模式」  分为互玩与捣蛋两种互动模式，减少主播互动压力
                        由主播设置游戏模式，观众可以根据模式筛选直播间
「新弹幕」      是对弹幕的扩充，在所有直播间中都适用
互动功能均设有冷却时间，或需要特权及蜜罐充值才能使用，
保证直播间的正常进行，在增加娱乐效果的同时，提升主播收益
共设计了四种捣蛋工具，适合FPS及MOBA类游戏，通过捣蛋特效
与主播互动，看主播的反应，增加节目效果
新弹幕形式通过选择快捷发送常用弹幕，出现[ 熊熊冒泡弹幕 ]
重复弹幕数量超过数额，触发[ 全屏弹幕 ]，节目效果爆棚！

[ 易选择 ]

通过用户选择热门弹幕选项，发送弹幕，降低弹幕输入门槛

[ 参与感 ]

单次气泡形式展现，达到数量触发全屏弹幕，增加参与感

[ 结合打赏 ]

新弹幕部分收费形式，让打赏具体化，降低打赏成本

[ 主播易读 ]

该形式也可以在主播端展现，便于主播及时关注观众状态

# 战后反馈
互动直播模式结束后，展示用户参与情况，等级提升情况，同时可以会看精彩瞬间
# 氪金方式
共有两种充值方式，蜜罐即金币充值，单次使用，特权时长充值，按时间使用
